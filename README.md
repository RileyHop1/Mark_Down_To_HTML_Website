# Project Overview


This projects goal is to take a MarkDown file and convert it into a basic html, file that can be displayed on the web. It uses the Rust programming langauge a robust programming language. 

# How to use

To convert a file from MarkDown to html all you need to is call the, convert_to_html() function with a file path
